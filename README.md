# Company CRM 


## Installation

- `composer install`
- `npm install`
- `php artisan key:generate`
- Edit `.env` and set your database connection details
- `php artisan migrate`
- `php artisan db:seed`
- `npm run dev`


## Usage
- php `artisan serve`

#### Development

```bash
# Build and watch
npm run watch

# Serve with hot reloading (not working)
npm run hot
```

#### Production

```bash
npm run production
```
